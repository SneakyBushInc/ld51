-------------------------------------------------
-                Cypurr Menace                  -
-------------------------------------------------

A Ludum Dare 51 by OddCreampuff & Iarwain.

Multiplayer co-op game, from 1 to 4 players.

Cheryl Bundy
Romain Killian

-------------------------------------------------

Controls:

Player1
-------
Gamepad #1
Move                    - Left Stick or D-Pad
Attack                  - Right Trigger/Bumper
Revive                  - Left Trigger/Bumper

Player2
-------
Gamepad #2

Player3
-------
Gamepad #3

Player4
-------
Gamepad #4

Menu
----
Key 1, 2, 3 or 4        - Select Player(s) Count
            .. or ..
A/Start on Gamepad #1   - 1P
A/Start on Gamepad #2   - 2P
A/Start on Gamepad #3   - 3P
A/Start on Gamepad #4   - 4P

Game Over
---------
Space/Start/Back        - Go back to Select Menu

Misc
----
F12 / Thumb Press       - Capture a screenshot
